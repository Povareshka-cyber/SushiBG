#Стек технологий
    
    React (Functional Components, Hooks)
    
    Tailwind CSS (Utility-first, Adaptive Design)
    
    React Router DOM
    
    Lucide React

    Vite

#Ключевые навыки

    State Management: работа с useState, useContext и Context API для управления корзиной и избранным.

    React Logic: фильтрация каталога, расчет итоговой суммы через .reduce(), управление количеством товаров (count).

    Navigation: программный редирект через useNavigate и работа с динамическими путями :slug.

    Data Handling: имитация работы с API через fetch и локальные JSON-файлы.

    Debugging: чтение ошибок консоли и устранение багов с типами данных.
